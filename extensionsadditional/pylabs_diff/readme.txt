Compares two sequences/sequence-like objects using a previously defined, dynamically
bound comparator method.

example application which
* Defines a dummy sequence class
* Builds to different sequences (instances of the Dummy class)
* Defines a comparator method
* Gets a sequence matcher
* Compares and displays the differences 

example application in 
https://bitbucket.org/incubaid/pylabs-core/src/tip/examples/pylabsdiff
