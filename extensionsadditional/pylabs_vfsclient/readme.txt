A client extension to the VFS server extension. It connects to 
a previously started VFS server over http/https and request 
information about the files/directories of the VFS mounted by the
server.

example application which
* Connects to a VFS server
* Gets the stats of a specified directory
* Gets the information of a specified file
* Lists the metadata version of the VFS mounted by the server

example application in 
https://bitbucket.org/incubaid/pylabs-core/src/tip/examples/pylabsvfsclient
